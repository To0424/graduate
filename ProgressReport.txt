GRADUATION — Tower Defense Game
Group 23 Progress Report
========================================

PROJECT OVERVIEW
----------------
"Graduation" is a 2D tower defense game where players take on the role of a Year 4 HKU student battling through course monsters across faculty buildings. Players clear levels to earn credits, aiming for 240 credits to graduate. The game features fixed-path enemy movement (Bloons TD style), multiple tower types, a skill tree system, and an overworld map of HKU campus.

Platform: PC (1920x1080) | Engine: Unity 2022.3 (URP 2D) | Language: C#


CURRENT PROGRESS
----------------
1. Core Architecture (Complete)
   - Game state machine (MainMenu, Overworld, Playing, Paused, LevelWon, LevelLost, Graduated)
   - Persistent singleton managers with DontDestroyOnLoad (GameManager, CreditManager, SkillPointManager, SkillTreeManager)
   - Scene flow: Main Menu → Overworld (faculty/level select) → Gameplay → back to Overworld
   - Save/load system via PlayerPrefs (completed courses, credits, skill points, skill tree)

2. Gameplay Systems (Complete)
   - Fixed waypoint path system with random path selection per difficulty tier (3 easy patterns created)
   - Wave spawner with multi-round support and round completion detection
   - Tower placement system with click-to-build on designated slots and preview ghost
   - 4 tower types: Rapid (cyan), Balanced (blue), Sniper (purple), Professor (gold)
   - Projectile system with homing and auto-destroy
   - Enemy pathfinding along waypoints with HP bars (green-to-red gradient)

3. Economy & Progression (Complete)
   - Per-level gold for tower purchases (resets each level)
   - Credits earned on first clear only — replaying completed courses gives no duplicate rewards
   - Skill points earned alongside credits (first clear only)
   - Skill tree with 4 sections (Social, Internship, Part-time Work, Certifications) buffing all towers
   - Professor tower unlock by clearing all courses in a faculty
   - New Game reset functionality (clears all progress)

4. UI & HUD (Complete)
   - Gameplay HUD: Gold, Lives, Round counter, Round timer, Enemies remaining
   - Tower shop panel (left side, color-coded buttons)
   - Pause/Resume overlay with Quit to Map
   - Level Won panel (shows rewards on first clear, "Already Completed" on replay)
   - Level Lost panel with Retry/Quit options
   - Graduation screen at 240 credits
   - Main Menu with Continue / New Game / Quit
   - Overworld with faculty buttons and level select

5. Auto-Setup & Tooling (Complete)
   - All scenes self-build at runtime (no manual Unity scene setup required)
   - Editor wizard tool (Graduation → Setup Starter Data) auto-generates all prefabs and ScriptableObjects
   - RuntimeSprite utility for procedural placeholder graphics
   - Architecture supports swapping in custom art via ScriptableObject sprite fields

Total scripts: 22+ across 8 folders (Core, Map, Enemies, Towers, Waves, Economy, UI, Editor)
